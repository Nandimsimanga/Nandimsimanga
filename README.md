## 🚀 Projects

### 🤖 AI-Powered Second-Hand Textbook Marketplace

> ASP.NET Core · Firestore · AI APIs · Semantic Search

A full-stack marketplace enabling students to buy and sell textbooks affordably, with AI-generated study notes, glossaries, and reference guides built in. Features include price suggestion, duplicate detection, semantic search, and OpenLibrary integration. Built an AI Study Assistant demo for the platform.

**Hackathon:** 🥉 3rd Place — IIE Varsity College Waterfall Hackathon 2025 *(Theme: Empowering Society with AI)*

-----

### 💰 AI Budget Tracker

> Kotlin · Android · RoomDB · AI APIs

A personal finance Android app built for people who don’t want to wrestle with budgeting details. An AI model tracks spending patterns, supports goal tracking, and automatically generates reports and graphs with actionable insights. Data is stored locally using RoomDB.

-----

### 🍽️ Recipe & Meal Planner

> .NET MAUI · Cross-platform

A cross-platform app for recipe storage and weekly meal planning. Supports full recipe CRUD, nutrition summaries and charts, grocery list generation, and automatic ingredient scaling.
